# 📊 Data Portfolio

## Présentation
Ce portfolio regroupe les différents projets data que j’ai réalisés dans le cadre de mon apprentissage et de ma pratique personnelle.

Je suis actuellement à la recherche d’un **stage dans le domaine de la data**, et ce dépôt a pour objectif de présenter les différentes activités et projets que j’ai réalisés.

## 📁 Contenu du repository

Ce portfolio contient plusieurs types de projets :

### 🤖 Machine Learning
Projets de modélisation et d’analyse de données utilisant différentes méthodes de machine learning.

### 📊 Data Visualisation
Projets de visualisation de données réalisés avec **Power BI**, comprenant des tableaux de bord interactifs permettant d’explorer et d’interpréter les données.

### 🔎 Analyse de données
Exploration, nettoyage et analyse de jeux de données afin d’extraire des informations pertinentes.

## 🛠 Outils et technologies

Les principaux outils utilisés dans ces projets sont :

- Python  
- Power BI  
- Pandas  
- Scikit-learn  
- Matplotlib  
- Seaborn  

## 🎯 Objectif du portfolio

Ce portfolio a pour objectif de démontrer :

- ma capacité à analyser et interpréter des données  
- ma maîtrise des outils de visualisation  
- ma compréhension des méthodes de machine learning  
- ma capacité à mener un projet data de bout en bout
